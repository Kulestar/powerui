Click and drag this element to get it to scroll


Note about inertial scrolling: Due to a patent held by Apple, we can't implement inertial scrolling. If you want it, see the source of DragToScroll.cs for notes on what to do.

This is done by C# methods which can be reused anywhere in the project. When this element is clicked, the C# script, DragScroller (which is in this example's files), stores the location of the mouse. onmousemove then updates the scroll value. As a side note, mouse events are triggered by touch events too.

So, the magic is in adding this:

onmousedown='DragScroller.StartScroll' onmousemove='DragScroller.ScrollNow'

But make sure you copy DragScroller.cs out of this example!
