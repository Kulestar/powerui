PowerUI inside a UnityUI
Note that it's not recommended to mix UI frameworks (to avoid confusion), but this example scene is here to show how it's done anyway!

This example uses code from the Drag to Scroll example to show that input works too, even if you rotate the element. Click and drag this to get it to scroll!



How to set this up


1. Create a RawImage node
2. Add a PowerUI -> HTML UI Panel script to it
3. Set the "Html File" property to a .html file
4. Done!


As with everything else, it's completely dynamic - you don't have to use a "static" html file. DOM access:

yourImage.GetComponent<HtmlUIPanel>().document

HtmlUIPanel.cs itself is fairly minimal so do take a look at the source of that too!
