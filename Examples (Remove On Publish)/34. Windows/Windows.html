The widget manager is a major new system in PowerUI 2. In short, it's the basis of a templating system for describing self-contained UI's.

For example, PowerUI 2 has built in support for context menu's. Context menu's vary in appearance enormously (and can occur for 3D elements too), so they've been built around the widget manager to be able to make (and share) highly customisable "widget templates" for UI components like these.
