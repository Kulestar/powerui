PowerSlide is used to declare sequences of things. It's great for animations and dialogue (speech between people).

This example (inspired by SimCity) shows PowerSlide being used to create a series of animations - check it out!



A quick note about competing selectors: If you have a selector "untriggering" whilst an animation is playing, it will cause a style refresh which 'wins' and drops the animation. For example, when an active selector no longer matches an element and it untriggers, an animation on that element may be cancelled. This will be getting attention in the future.
Top Tip - Take a look at the wiki articles!
PowerSlide can be used in a variety of ways, and we're also working on a free, open source editor to generate the json files that it uses too.
Take a look at PowerSlideExample.json - the file that actually runs - and AnnotatedPowerSlideExample.json - which describes what's going on.
